Quōmodo dē Quālitāte Interrogēmus (Figūrae) = Ars imitātur nātūram — “Art imitates nature” — Aristotle

Hic est circulus = This is a circle
Circulus figūra rotunda est = A circle is round

Hoc est triangulum = This is a triangle
Triangulum figūra angulāta est = A triangle is angled

Hoc est quadrātum = This is a square
Quadrātum figūra aequa est = A square is equal

Haec est stēlla = This is a star
Stēlla figūra pulchra est = A star is beautiful
