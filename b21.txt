Estne (Is it...?) = "Is it to be emperor?" —Augustine
Estne līber in bulgā? = Is the book in the bag?
Est! = It is!
Certē! = Certainly!
Minimē! = No!
Nequāquam! = Not at all!
Nonnē est calamus in bulgā? = Isn’t the pen in the bag?
Ita est! = Yes, it is!
Est, certē! = It is, certainly!
Num magistra es? = You aren’t the teacher, are you?
Nōn sum. = I am not.
Estne calamus / līber illīc? = Is the pen / book over there?
Calamī / lībrī sunt hīc. = The pens / books are here.
Da mihi hunc calamum / librum. = Give me this pen / book.
Videō mēnsam / fēlem. = I see the table / cat.
Ubī est illa mēnsa / ille fēlēs? = Where is that table / cat?
Illīc est! = It is there!
Hīc est! = It is here!
