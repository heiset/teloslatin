Perterritus et Adflīctus (Terrified and Distressed) = Miserere miserōrum — “Have pity on the miserable” — proverb
Hic puer perterritus est = This boy is terrified
Hunc puerum vidēmus esse perterritum = We see this boy to be terrified
Ō tē miserum = Oh poor you
Vidēō hunc virum esse adflīctum = I see this man to be distressed
Hic est vir adflīctus = This man is distressed
Puerī adflīctī lacrimās videō = I see the tears of distressed boys
Ō illum miserum = Oh that poor boy
Haec est puella perterrita = This girl is terrified
Hanc puellam videō perterritam esse = I see this girl to be terrified
Puellae perterritae faciem videō esse perterritam = I see the face of the terrified girl to be terrified
Ō miseram puellam = Oh poor girl
Philippus miserētur puellae perterritae = Philippus pities the terrified girl
